jekyll = 打造静态网站或者博客站点


1） _config.yml : 修改_config.xml,把一些信息修改为自己的
2）index.html : 是网站的首页入口
3）img 文件夹下是项目中使用的图片
4）



jekyll 的目录结构介绍

-> _config.yml  : 存储的是 配置 信息
-> _includes
-> _layouts ：layouts是包裹在文章外部的模板
-> _posts：  这里放的就是你的文章了。文件格式很重要，必须符合：`YEAR-MONTH-DAY-TITLE.MARKUP`。
-> index.html : 首页设置
